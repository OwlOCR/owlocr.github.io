How It Works

1. Capture or Select

Grab a screen area with Cmd + F1, drop a PDF/image, or use Finder Extension. iPhone/iPad scans import seamlessly.

2. Recognize On-Device

High-quality OCR runs locally. Multi-language support: English, French, German, Spanish, more on macOS 11+.

3. Copy or Export

Text goes straight to clipboard. Or export as plain text file or searchable PDF with preserved layers.

4. Customize and Repeat

Tweak shortcuts, dictionary, wrapping. Process batches via directories for efficiency.

FAQ

Is my data secure?

Can I use it from anywhere?

Yes. Global shortcuts (Cmd + F1/F2) work app-wide. Finder Extensions for files. Menu bar/Dock options.

What file types work?

PDF, GIF, PNG, JPEG/JPG. Batch process directories or pages.
